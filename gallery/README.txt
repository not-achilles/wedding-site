========================================================================
HOW TO ADD YOUR PHOTOS & VIDEOS TO THE GALLERY
========================================================================

The website is configured to automatically scan this directory and load
your wedding media. You do NOT need to edit any code.

Simply follow these steps:

1. Copy your photos and videos into this folder ("gallery/").
2. Rename your files numerically:
   - For photos, name them:
     1.jpg  (or 1.jpeg, 1.png, 1.webp)
     2.jpg
     3.jpg
     ... up to 30.jpg

   - For videos, name them:
     1.mp4  (or 1.webm, 1.mov)
     2.mp4
     3.mp4
     ... up to 30.mp4

3. Refresh your website (index.html) in the browser.

The site will automatically detect and display whatever files you have named
numerically in this folder, with interactive hover playbacks and fullscreen Lightbox modal!

Note: If this folder is empty, the website will display default Indian
wedding placeholders so it never looks blank.
